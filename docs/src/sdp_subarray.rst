SDP Subarray Device
===================


Introduction
------------

The interface description for the SDPSubarray device being developed in PI#3
can be found on `SKA Confluence
<https://confluence.skatelescope.org/pages/viewpage.action?pageId=74716479>`_.

The SDPSubarray device is published as a Docker image, available at:
https://hub.docker.com/r/skaorca/tangods_sdp_master

The SDPSubarray device respects to environment variables as feature toggles:

- TOGGLE_CONFIG_DB
- TOGGLE_CBF_OUTPUT_LINK
